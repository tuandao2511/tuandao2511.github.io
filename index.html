Help

Did you know that some phones (like Samsung or Sony for example) have a PowerSaving Mode that might stop alarms from working? Not to miss your alarms, turn off Power Saving mode or add Smart Alarm into your whitelist. Also, if you use third-party memory cleaners, make sure they do not unload the application from memory

Whitelist

Here we have the instructions to add the appinto whitelist for specific manufactures

  1. Go to Settings -> Power Saving
  2. See if you can add Alarm Study to the protected whitelist. Also, if your device has a memory manager (tap a right square button) you need to lock the app. Just click on the lock icon. If you do not have such settings, just ignore it.
  1. Use the Smart Manager app, the default or go to Settings -> Battery or Power Saving Mode
  2. Disable Alarm Study from Power Saving mode.
  1. Go to default Security app -> Permissions -> Autostart and enabled Alarm Study on the list
  2. Lock the app in RAM. Open memory manager (tap on left button) scroll down by the app icon. The lock icon will appear at theapp icon.
  1. Go to Settings -> Power management -> then tap STAMINA mode
  2. Activate STAMINA mode if it is deactivated
  3. Make sure that the Extended standby feature is turned on, then tap Apps active in standby -> Add application
  4. Swipe left or right to view all applications and services, then mark the checkbox for Alarm Study app
  5. When you finished, tap Done
